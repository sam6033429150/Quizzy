# Quizzy

Quizzy is a web-based quiz application that allows users to register, log in, take quizzes, track their scores, and view rankings through a leaderboard system. The project was developed using HTML, CSS, and JavaScript with browser Local Storage for data persistence.

---

## ✨ Features

### User Authentication

* User Registration
* User Login
* Session Management using Local Storage
* Welcome User Dashboard

### Dashboard

* User-friendly interface
* Navigation sidebar
* Statistics cards
* Quick access to quizzes and leaderboard

### Quiz System

* Multiple-choice questions
* Score calculation
* Quiz timer
* Progress tracking

### Leaderboard

* Displays player rankings
* Tracks quiz scores
* Dynamic leaderboard updates

### Responsive Design

* Mobile-friendly layouts
* Responsive forms
* Adaptive dashboard components
* Optimized user experience across devices

## 🚀 How to Run the Project

### Option 1: Using VS Code Live Server (Recommended)

1. Open the project folder in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.
5. The application will open automatically in your browser.

---

### Option 2: Open Directly in Browser

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html`.
4. The application will launch in your default browser.

---

## 👤 User Workflow

1. Register a new account.
2. Log in using your credentials.
3. Access the dashboard.
4. Start a quiz.
5. Complete the quiz and receive a score.
6. View rankings on the leaderboard.

---

## 💾 Data Storage

This project uses the browser's **Local Storage** to store:

* User Information
* Login Credentials
* Active User Session
* Quiz Scores
* Leaderboard Data

**Note:** Data is stored locally in the browser and is not connected to a database.
